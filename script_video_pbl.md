# NASKAH BACAAN (SCRIPT VOICE OVER) VIDEO PBL
**Website Program Studi D3 Teknik Listrik - Politeknik Negeri Manado**

Naskah ini berisi teks ucapan (apa yang akan dikatakan) oleh narator/presenter selama video presentasi Project Based Learning (PBL). Naskah dibuat ringkas, lugas, dan profesional untuk dibaca saat rekaman suara.

---

### 1. Opening / Judul Proyek (Estimasi Durasi: 1-2 menit)
**Narator:**
> "Halo semuanya, salam sejahtera bagi kita semua. Selamat datang dalam pemaparan Project Based Learning kami. 
> 
> Pada kesempatan kali ini, kami dengan bangga mempersembahkan hasil proyek pengembangan Website Resmi Program Studi D3 Teknik Listrik Politeknik Negeri Manado.
> 
> Kami dari tim PBL mahasiswa D3 Teknik Listrik, didampingi oleh dosen pembimbing kami, berkolaborasi untuk merancang dan membangun platform digital ini sebagai solusi modern bagi kebutuhan informasi akademik di lingkungan kampus kami."

---

### 2. Latar Belakang Masalah (Estimasi Durasi: 2-3 menit)
**Narator:**
> "Sebagai salah satu program studi aktif di Politeknik Negeri Manado, D3 Teknik Listrik membutuhkan media penyebaran informasi yang cepat dan mudah diakses. Namun, selama ini kami menghadapi beberapa kendala nyata di lapangan.
> 
> Pertama, penyebaran informasi penting seperti jadwal perkuliahan masih sering dilakukan secara manual atau tidak terpusat. Kedua, mahasiswa maupun masyarakat luar cukup kesulitan untuk mengakses data kurikulum terbaru serta profil keahlian para dosen kami secara daring. Dan ketiga, belum tersedianya portal informasi terpadu yang ramah pengguna dan responsif di perangkat mobile. 
> 
> Oleh karena itu, digitalisasi sistem informasi ini menjadi langkah mendesak yang harus segera kami realisasikan."

---

### 3. Tujuan Proyek (Estimasi Durasi: 1 menit)
**Narator:**
> "Melalui proyek PBL ini, kami memiliki beberapa tujuan utama. 
> 
> Pertama, membangun website resmi program studi yang modern, interaktif, dan mudah diakses dari perangkat apa saja. 
> 
> Kedua, menyajikan data akademik secara transparan, mulai dari visi misi, profil dosen, kurikulum, hingga fasilitas laboratorium.

> Dan ketiga, menyediakan panel admin dinamis berbasis database agar staf administrasi prodi dapat memperbarui informasi secara instan tanpa perlu menyentuh baris kode."

### 4. Mitra / Stakeholder (Estimasi Durasi: 1 menit)
**Narator:**
> "Stakeholder utama dari proyek ini adalah seluruh civitas akademika D3 Teknik Listrik Politeknik Negeri Manado, yang meliputi dosen, staf administrasi, serta mahasiswa aktif. 
> 
> Selain itu, website ini juga dirancang sebagai pintu informasi tepercaya bagi pihak eksternal, seperti calon mahasiswa baru yang ingin mendaftar, orang tua mahasiswa, serta mitra industri yang ingin melihat kompetensi lulusan dan fasilitas laboratorium kami."

---

### 5. Proses Pengerjaan (Estimasi Durasi: 3-4 menit)
**Narator:**
> "Proses pengerjaan proyek ini kami lalui dalam empat tahapan utama.
> 
> Tahap pertama adalah observasi dan analisis kebutuhan. Kami mengumpulkan data langsung terkait silabus kurikulum, keahlian dosen, dan daftar inventaris laboratorium kelistrikan.
> 
> Tahap kedua adalah perancangan UI/UX. Kami mendesain antarmuka website menggunakan Figma untuk memastikan navigasi terasa nyaman, intuitif, dan responsif.
> 
> Tahap ketiga adalah coding dan pengembangan sistem. Kami menulis baris kode untuk antarmuka depan, membuat rute API untuk pertukaran data, serta merancang database relasional untuk menyimpan semua informasi secara aman.
> 
> Tahap keempat adalah pengujian dan peluncuran. Kami melakukan uji coba fungsi web, mengoptimalkan kecepatan akses, dan melakukan deployment pada server agar dapat diakses oleh publik."

---

### 6. Teknologi yang Digunakan (Estimasi Durasi: 2-3 menit)
**Narator:**
> "Untuk menghasilkan website dengan performa optimal dan aman, kami menggunakan kombinasi teknologi modern:
> 
> Di bagian frontend, kami memilih Next.js berbasis React untuk menyajikan rendering antarmuka yang cepat dan interaktif. Sedangkan untuk sisi backend, kami membangun web service API terpisah menggunakan framework Ktor dengan bahasa Kotlin untuk menangani proses data secara asinkron dan efisien.
> 
> Untuk tata letak dan desain visual, kami menggunakan Tailwind CSS yang menghasilkan antarmuka responsif dan estetik di layar handphone maupun komputer.
> 
> Di bagian database, kami mengandalkan PostgreSQL yang terinfrastruktur secara cloud di Supabase, yang terhubung langsung dengan backend Ktor kami. Dan untuk keamanan akses panel admin, kami mengintegrasikan NextAuth.js agar data sensitif program studi tetap terlindungi dengan aman."
