STRUKTUR VIDEO PBL

1. Opening / Judul Proyek (1–2 menit)
Tampilkan:
•	nama proyek 
•	nama Mahasiswa 
•	program studi 
•	kampus Politeknik Negeri Manado 
•	logo kampus 
•	tahun akademik 
Contoh narasi:
“Project Based Learning Sistem Digital Pengelolaan Keuangan UMKM berbasis web dan mobile.”

2. Latar Belakang Masalah (2–3 menit)
Jelaskan masalah nyata yang diangkat
Contoh:
•	UMKM masih manual 
•	nelayan sulit pencatatan hasil tangkap 
•	koperasi masih menggunakan buku kas 

3. Tujuan Proyek (1 menit)
Menjelaskan apa yang dibangun.
Contoh:
•	membangun aplikasi 
•	meningkatkan efisiensi 
•	digitalisasi proses bisnis 

4. Mitra/Stakeholder (1 menit)
Jika ada mitra:
•	UMKM 
•	koperasi 
•	nelayan 
•	instansi pemerintah 
Tampilkan lokasi mitra bila ada.

5. Proses Pengerjaan (3-4 menit)
Tunjukkan tahapan proyek:
•	observasi 
•	analisis kebutuhan 
•	desain UI/UX 
•	coding 
•	testing 
•	deployment 
Bisa tampilkan diagram singkat.

6. Teknologi yang Digunakan (2-3 menit)
Contoh:
•	Next.js 
•	React 
•	Flutter 
•	MySQL 
•	PostgreSQL 
•	Laravel 

7. Demo Sistem / Produk (bagian paling penting) (3–4 menit)
Tampilkan:
•	login 
•	dashboard 
•	fitur utama 
•	input data 
•	output laporan 
•	notifikasi 
•	web app demo 
Ini bagian terpanjang.

8. Hasil Implementasi / Dampak (1-2 menit)
Contoh:
•	efisiensi waktu naik 
•	pencatatan lebih akurat 
•	membantu mitra 

9. Dokumentasi Tim (1-2 menit)
Tampilkan:
•	diskusi tim 
•	coding bersama 
•	kunjungan lapangan 
•	presentasi ke mitra 

10. Closing (30–60 detik)
Kesimpulan singkat + ucapan terima kasih.
